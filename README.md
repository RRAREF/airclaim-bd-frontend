# AirClaim BD

AirClaim BD is a Smart Airport Lost & Found Management System for Bangladesh airports.

## Features
- User Registration
- User Login
- Report Lost Item
- Report Found Item
- Airport Selection
- Bag Tag Code Verification
- Ticket Number Verification

## Technologies
- React.js
- JavaScript
- HTML5
- CSS3
- Bootstrap
- Git
- GitHub
- Vercel
